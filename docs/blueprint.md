# **App Name**: EthicalAI

## Core Features:

- Natural Language Mission Intake: A sophisticated command center where users describe complex AI missions using natural language, serving as the primary intake tool for the agentic workflow.
- Mission Intelligence Preview: Real-time updates that reflect mission parameters like estimated agents and duration based on textarea input analysis.
- Ethical Policy Pre-Check Tool: A safety-first tool that reasons through the proposed mission intent to ensure alignment with defined ethical boundaries and compliance rules before launch.
- Intelligent Intent Detection: An AI-powered mechanism to automatically identify the underlying mission objective and categorize it for processing.
- Premium Multi-Modal Uploads: Context-aware UI actions to attach files, import datasets, or record voice input directly into the mission scope.
- Immersive Sequenced Onboarding: A cinematic dark-mode loading screen with pulsed wine gradients and staggered letter animations to establish brand authority.

## Style Guidelines:

- A premium dark palette anchored in #0d1117 for the background, with a sophisticated wine primary accent (#810B38) for energy and authority.
- Neutral secondary elements in #DCC3AA (bone/beige) and surface highlights in #F1E2D1 create high-end legibility against deep crimson sub-tones.
- Recommendation: 'Inter' used across all scales (600 weight for wordmarks, 700 for headlines, and 400 for body) to maintain a precise, objective, and neutral tone. Note: currently only Google Fonts are supported.
- Complex interaction feedback using Framer Motion: letter-by-letter staggering for loading, vertical layout shifts on mount, and spring-based hovering for tactile button responsiveness.
- High-focus centered command area with a lateral 'Mission Intelligence' preview card that transitions seamlessly from mobile stacks to desktop side-bars.
- Minimalist iconography featuring high-contrast wireframes for settings, files, and voice input, utilizing semi-transparent borders for depth.
