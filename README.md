# EthicalAI — Autonomous Intelligence OS

EthicalAI is a production-grade autonomous intelligence platform governed by a Constitutional AI framework. This project is organized into three core layers:

- **Nexus**: The intelligence orchestration layer (Intake, Planning, Agents).
- **Sovereign**: The governance and policy enforcement layer (Ethics, Audit, Compliance).
- **Citadel**: The security and defense layer (Threat Detection, Sandboxing, Identity).

## Project Structure

```text
├── nexus/               # Intelligence Orchestration
├── sovereign/           # Governance & Policy
├── citadel/             # Security & Defense
├── core/                # Shared AI & System Utilities
├── frontend/            # Next.js UI (located in src/app)
├── docs/                # Architecture & API Specs
└── scripts/             # Operational & Security Scripts
```

## Getting Started

1.  **AI Dev Mode**: `npm run genkit:dev`
2.  **UI Dev Mode**: `npm run dev`
